# Angular Hospital Management System

A responsive hospital management frontend built with Angular.

## 🚀 Features
- Patient registration
- Medicine inventory
- Billing system
- Live search dropdowns
- REST API integration

## 🛠 Tech Stack
- Angular
- TypeScript
- Bootstrap
- RxJS
- REST APIs

## ▶ Setup Instructions
1. Clone the repo
2. Run: npm install
3. Run: ng serve
4. Open: http://localhost:4200
